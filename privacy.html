Política de Privacidade

Última atualização: 2026-01-31

Esta Política de Privacidade descreve como uma aplicação privada de automação, executada no n8n (Community Edition), utiliza permissões OAuth do Google para aceder a dados do proprietário da conta, com a finalidade exclusiva de executar automações pessoais.

1) Quem é o responsável

Esta aplicação é utilizada de forma privada pelo proprietário da conta Google. Não existe disponibilização pública do serviço, nem registo de utilizadores.

2) Dados que podem ser acedidos

Dependendo das permissões concedidas (scopes) no ecrã de consentimento, a aplicação pode aceder a dados de serviços Google como Gmail, Google Drive e Google Calendar, apenas na conta do proprietário.

3) Finalidade do acesso

  • Automatizar tarefas (ex.: ler emails para extrair informação, guardar anexos, criar eventos, organizar ficheiros).
  • Enviar notificações e gerar relatórios a partir de dados autorizados.

4) Partilha de dados

Esta aplicação não vende, não aluga e não partilha dados com terceiros. Os dados são utilizados apenas para executar os workflows configurados no n8n pelo proprietário.

5) Armazenamento e retenção

As credenciais OAuth (tokens) são armazenadas no ambiente onde o n8n está hospedado. Dados processados podem ser armazenados localmente conforme a configuração dos workflows (por exemplo, logs, ficheiros ou bases de dados).

6) Segurança

O acesso ao n8n é protegido por mecanismos de autenticação e pelo controlo do ambiente de hospedagem. Recomenda-se manter o servidor atualizado, usar HTTPS e restringir o acesso à interface administrativa.

7) Revogação de permissões

O proprietário pode revogar o acesso a qualquer momento através das definições da Conta Google (Segurança → Acesso de terceiros), removendo a aplicação. Após a revogação, os tokens deixam de funcionar.

8) Contacto

Para questões relacionadas com esta Política, utilize o contacto definido no ecrã de consentimento do Google.
